#### **加速器** <a href="#jia-su-qi" id="jia-su-qi"></a>

> **禁止使用免费加速器**
>
> **雷神/薄荷/鲜牛等加速器还有所谓的`**加速器盒子`自行尝试,不行就换下列加速器**
>
> > **UU加速 `路由模式/模式3`**
> >
> > **迅游加速器 `模式3`**
> >
> > **奇游加速器 `模式1`**
> >
> > **3A加速器 `加速瓦洛兰特 全区服模式`**
>
> _**尝试更换加速器 更换加速器模式 更换加速器节点 直到能注入使用为止**_
